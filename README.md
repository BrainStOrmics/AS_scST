# AS_scST

Code and analysis repository for “Spatiotemporal Control of Vascular Cell Plasticity during Atherosclerosis”

---

## Overview

- 00_Data_preprocessing: Preprocessing workflows for raw spatial transcriptomic and single-cell datasets.
- Figure_1-4: Demo datasets and original analysis pipelines used to generate main Figures 1–4.
- Figure_1-4_updated (fig1-4): Revised and extended analysis scripts for main Figures 1–4.
- Supplementary_Figure_1_6 (sup1-6): Analysis pipelines for Supplementary Figures 1–6.
